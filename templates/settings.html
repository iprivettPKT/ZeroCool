Remote access

ZeroCool's UI is bound to 127.0.0.1 (it runs commands as root — never expose it on a network). Reach it from another machine by forwarding the port over SSH.

1 · SSH tunnel

Run this on your Mac / client, then browse to http://127.0.0.1:{{ port }}.

2 · One-command connect script

Keep this on your Mac. It opens the tunnel and your browser: ./zerocool-connect.sh [-i keyfile] user@box

Download zerocool-connect.sh

3 · Auto-start on the headless box

Install once so ZeroCool always runs — no more SSH-in-and-launch.

Download zerocool.service
sudo cp zerocool.service /etc/systemd/system/ && sudo systemctl daemon-reload && sudo systemctl enable --now zerocool
